# GMAD: Great Magento Architecture Decoupling

The open-source project to decouple, modernize, and slim down Magento 2.

**221 modules scanned. 121 removable. 12 parallel workstreams.**

Read the full documentation at **[gmad.qoliber.com](https://gmad.qoliber.com)**

## Quick Start

- [Implementation Plan](https://gmad.qoliber.com/IMPLEMENTATION_PLAN/) - 12 workstreams, 46 tasks, step-by-step
- [Module Disable Groups](https://gmad.qoliber.com/disable_groups/) - Copy-paste commands to slim down your Magento
- [Decoupling Reports](https://gmad.qoliber.com/decoupling/configurable_product/) - Deep analysis of every module

## Contributing

1. Pick a workstream from the [Implementation Plan](https://gmad.qoliber.com/IMPLEMENTATION_PLAN/)
2. Follow the 4 established patterns (documented with code examples)
3. Submit a PR
4. Run the verification checklist
